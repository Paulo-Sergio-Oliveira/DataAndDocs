# DataAndDocs

Este repositório contém uma coleção de dados e documentos utlizados no estudo "Usando LLMs em Especificações de Requisitos de Software: Uma Replicação do Estudo". Eles estão organizados nas seguintes categorias:

## Conteúdo

- **Dados:** Esta pasta contém conjuntos de dados brutos e processados utilizados no estudo.
- **Geração:** Esta pasta abriga os documentos de ERS gerados por LLMs e o Benchmark Humano.
- **Prompts:** Esta pasta contém os prompts utilizados no processo de geração e validação.
- **Validação e Correção:** Esta pasta abriga os documentos de Validação e Correção gerados por LLMs e o Benchmark Humano.
